## Why Use This

The Apple Search Ads dashboard is powerful, but extracting meaningful performance data across campaigns, keywords, and ad groups requires tedious exporting and manual analysis. This MCP server cuts through all of that.

**What you get:**

- **Instant app campaign analytics** — ask about installs, TTR, CPI, and conversions without navigating the Apple Search Ads dashboard.
- **No setup complexity** — no Apple Search Ads API certificate management required; connect through Two Minute Reports in minutes.
- **Multi-account support** — analyze multiple Apple Search Ads accounts from a single conversation.
- **Cross-platform context** — combine with 22+ other sources like Google Analytics 4, Google Ads, and Meta Ads to compare your app acquisition channels side by side.
